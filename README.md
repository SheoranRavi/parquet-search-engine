# Search engine for parquet files

## Frontend
- Create .env file and add the VITE_API_URL to it with `/api` suffix
- In dev environment can be run using `npm run dev`

## Backend
- Set the env variable PORT if you want to run it on a specific port. Default is 9080.
- Run `make build` to build it.
- Then run from `backend/bin/parquet-api`.
